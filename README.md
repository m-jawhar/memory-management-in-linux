# Memory Management in Linux

## Overview
This repository contains self-learning practise on operating system concepts, focusing on **memory management in Linux**.

## Topics Covered
1. **Virtual Memory in Linux**  
   - Explanation of how Linux uses virtual memory.  
   - Key concepts: address space, paging, swapping.  

2. **Demand Paging Demonstration**  
   - Implementation using a memory-intensive program.  
   - Example code snippet in C.  
   - Steps to run and observe behavior.  

3. **Page Fault Analysis with vmstat**  
   - Using `vmstat` to monitor page faults.  
   - Comparing results before and after running the program.  
   - Interpretation of output.
